Política de Privacidad

DriveZone MX — Última actualización: julio 2026

Esta aplicación es una herramienta de uso interno de DriveZone MX para automatizar la publicación de contenido propio en redes sociales, incluyendo TikTok. Esta política explica cómo se maneja la información en relación con el uso de la API de TikTok.

1. Información que se procesa

La aplicación procesa únicamente contenido propio de DriveZone MX (fotos y videos de vehículos en inventario, junto con descripciones de texto generadas para las publicaciones). No se recopila, almacena ni comparte información personal de usuarios de TikTok ni de terceros.

2. Uso de la API de TikTok

Esta aplicación utiliza el Content Posting API de TikTok exclusivamente para publicar contenido en la cuenta oficial de DriveZone MX en TikTok. El acceso está limitado a esa cuenta y no se utiliza para acceder a datos de otros usuarios de la plataforma.

3. Almacenamiento de datos

Los tokens de acceso y credenciales necesarias para el funcionamiento de la aplicación se almacenan de forma segura y no se comparten con terceros.
